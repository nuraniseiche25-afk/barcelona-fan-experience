# BARCELONA FAN EXPERIENCE

Web Client Application berbasis React JS dengan konsep cinematic football experience untuk penggemar FC Barcelona.

## Features
- Cinematic Landing Page
- Login Experience
- Chant Audio
- Player Spotlight
- Trophy Room
- Iconic Moments
- Camp Nou Virtual Section

## Tech Stack
- React JS
- JavaScript
- Tailwind CSS
- Vite
